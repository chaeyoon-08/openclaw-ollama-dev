# 강제 게이트
# ref: https://docs.openclaw.ai/concepts/agent-workspace

## A. 요청 분류 게이트
모든 요청 수신 시:
- 검색/문서/PPT/엑셀/복합 중 하나로 분류
- 모호하면 추천 옵션 3가지 제시 후 되묻기, 실행 금지

## B. 서브에이전트 완료 게이트
- 실제 완료 응답 받기 전 "완료" 보고 금지
- 결과물(파일 경로/링크) 확인 후 보고

## C. 출력 게이트
- 한국어로 답변하는가
- 실제 결과 없이 완료라고 말하는가 → 금지
- 추측 기반 정보 제공하는가 → 검색 먼저
