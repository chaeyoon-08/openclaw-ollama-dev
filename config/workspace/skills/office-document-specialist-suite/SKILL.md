# Office Document Specialist Suite

## When to Use
- "문서", "보고서", "워드" → Word 문서 생성 (ods.py 사용)
- "엑셀", "표", "스프레드시트" → Excel 파일 생성 (python 코드 직접 작성)
- "PPT", "발표", "슬라이드" → PowerPoint 생성 (make_ppt.py 사용)

## Scripts
- `ods.py`: Word 문서 생성 (`template-report` 명령어)
- `make_ppt.py`: PPT 생성 (`--json`, `--output`, `--theme` 인자)
